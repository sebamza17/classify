Cards

Cards are versatile components with 100% width that can fit wherever they are. It implements a title, image and content and has a hover effect with z-depth manipulation.

Has a .padding-half applied. Also, can truncate text adding .text-truncate class

Card image

Card image has a fixed ratio of 4:3, but it can be changed using .image-ratio variations on the image container. The image itself is implemented via background-size: cover to avoid weird cuts or overflows on img tags

Card content

Card content is a simple container with .padding-full class applied. It has no height constrain and it can be used to display text or links
